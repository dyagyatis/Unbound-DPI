# Unbound 🛡️

**Unbound** — это компактный и быстрый лаунчер для обхода сетевых ограничений. Проект объединяет в себе мощный движок для борьбы с DPI (на базе Zapret) и встроенный Telegram MTProto прокси.

### 🚀 Основные возможности
* **Автономность**: Программа не требует доступа к сети для работы (все зависимости локальны).
* **Безопасность**: Использует проверенные бинарники `winws2` для работы с сетью.
* **Anti-detect**: Возможность подключения личных Cloudflare Workers для защиты от бана в Telegram.
* **Удобство**: Управление списком заблокированных ресурсов (`reestr_hostname.txt`) прямо из интерфейса.

### 🛠 Установка
1. Скачайте актуальную версию.
2. Распакуйте в удобное место.
3. **Запустите `Unbound.exe` от имени администратора** (необходимо для работы драйвера WinDivert).

### ⚙️ Настройка прокси
Для работы прокси-сервера Telegram:
1. Вкладка **Сервисы** -> **Telegram MTProto Прокси**.
2. Укажите желаемый порт (по умолчанию `1080`).
3. (Опционально) Укажите домен вашего Cloudflare Worker для скрытия трафика.
4. Нажмите «Запустить».

### 📋 Структура папок (для разработчиков)
Для корректной сборки проекта в папке `binaries` должны находиться:
- `winws2.exe`
- `TgWsProxy_windows.exe`
- Папка `files` с файлом `reestr_hostname.txt`
