# Unbound DPI

**Unbound DPI** — это графическая оболочка для [zapret2](https://github.com/bol-van/zapret2), созданная для удобного обхода ограничений DPI на Windows. 

Проект создан, чтобы сделать управление сетевыми стратегиями простым, наглядным и быстрым.

## Основные возможности
*   **Быстрое переключение стратегий:** Выбирай готовые пресеты для YouTube, Discord и других сервисов в один клик.
*   **Визуализация работы:** Отслеживай статус процесса в удобном интерфейсе.
*   **Легковесность:** Написано на C# (WPF), использует минимум ресурсов.
*   **Автоматизация:** Управление жизненным циклом `winws2.exe` — запуск, перезапуск и завершение без лишних окон терминала.

## Установка
1. Скачай [последний релиз](ссылка_на_релиз).
2. Распакуй архив в удобную папку.
3. Запусти `Unbound.exe` **от имени администратора**.

## Как это работает?
Unbound является оболочкой (GUI-враппером) для `winws2.exe` (от проекта zapret2). Он берет на себя рутину по передаче параметров командной строки и управлению правами доступа, предоставляя тебе удобный интерфейс.
